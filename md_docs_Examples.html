Examples

Directory examples contains one example to show the workflow of ExaGeoStat zgen_mle_test.c.

Important notice: ExaGeoStat packages has been designed to eventually support both exact and approximate computations of Maximum Likelihood Evaluation (MLE) problem. For now, only exact computation is supported.
